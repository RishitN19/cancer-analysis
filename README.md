## How to reproduce
  1. Request SEER access at seer.cancer.gov/data
  2. Download TCGA-BRCA clinical data at portal.gdc.cancer.gov
  3. Install dependencies: 'pip install pandas numpy matplotlib seaborn lifelines scikit-learn'
  4. Run notebooks in order: 01_tcga_exploration, 02_seer_exploration, 03_ml_model
